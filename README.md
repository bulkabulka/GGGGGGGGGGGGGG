# 🎲 Bulka Casino Pro
# Autorzy: Vlad Kelman, Margo Rybchenko


**Opis projektu:**
Interaktywna platforma kasynowa stworzona w celach edukacyjnych. Projekt demonstracyjny pokazujący logikę gier hazardowych i zarządzanie danymi w przeglądarce.

**Funkcje:**
- **Automaty:** 5 różnych motywów (w tym ukryte Easter Eggs).
- **Blackjack:** Gra z krupierem i systemem liczenia kart.
- **Portfel:** System wpłat (BLIK/Karta) i obsługa kodów promocyjnych.
- **Profil:** Statystyki gracza i automatyczny zapis salda przez `localStorage`.

**Technologie:**
- JavaScript (Vanilla)
- HTML5 / CSS3
